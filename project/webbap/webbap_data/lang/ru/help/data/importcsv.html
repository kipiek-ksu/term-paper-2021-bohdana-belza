Импорт из CSV файла

CSV-формат содержит значения, разделенные запятыми.

Предусмотренный формат файла - простой текстовый файл со списком имен полей. В следующих строках размещаются данные, одна запись на строку.

Разделитель полей по-умолчанию - запятая, каждое поле в записи обрамляется запятыми.

Записи разделяются переходом на новую строку (нажатием кнопки Enter в текстовом редакторе).

Пример файла:

  имя,рост,вес
  Сергей,180см,80кг
  Андрей,170см,60кг
  Борис,190см,20кг

Внимание: должны быть соблюдены типы соответствующих полей.
