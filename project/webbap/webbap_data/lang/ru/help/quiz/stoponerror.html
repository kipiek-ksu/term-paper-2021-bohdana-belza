Остановка при ошибке

Вопросы импортируются в два этапа. Сначала импортируемый файл проверяется на правильность. Затем идет запись в базу данных. Если этот параметр выбран (это значение по умолчанию) и ошибка была обнаружена при проходе синтаксического анализа, то запись в базу данных не производится.

Если эта опция не включена, то будет сделана попытка записать любые допустимые вопросы в базу данных.
