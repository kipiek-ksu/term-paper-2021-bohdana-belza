Показать лучшие результаты

Отображение лучших для этого урока результатов. Студенты получившие высокие оценки могут выбрать для себя имя в этом списке. Имя будет проверено фильтром 'badwords' для выявления непреемлимых значений.

Список лучших результатов не отображается если включена опция 'Тренировочный урок'.
