Ограничение времени

Эта установка определяет предельную продолжительность урока. Студенты будут видеть таймер на javaScript и время будет записано в базу данных. Из-за особенностей javaScript таймер не выбросит студента в случае превышения времени, тем ме менее ответы на вопросы после лимита времени не будут учтены. Ограничение времени проверяется в базе данных когда студент отвечает на вопросы.
