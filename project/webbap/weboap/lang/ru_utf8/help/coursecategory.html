Категории курсов

Администратор сайта может назначить категории для разделения курсов.

Например: "Факультеты", "Кафедры", "Дисциплины" и т.д.

Выберите категорию, которая наиболее близко подходит для вашего курса. Это повлияет на место отображения курса в общем списке (Вашим ученикам будет проще его найти).
