Qui si può abilitare il sistema di colorazione sulla base della frequenza di click

Se la funzionalità viene attivata, i tag verranno colorati sulla base del numero di click ricevuti. Più il tag diventa rosso, più è popolare.

Esempio

Quando la funzionalità è disattivata, Text Clouds disegnerà i tags tutti neri

footer html moodle document

Abilitando la funzionalità, ogni click sui tags viene contato, e tutti i tags verranno colorati in base al numero di click ricevuti.
