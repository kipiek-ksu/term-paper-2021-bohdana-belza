Опции вопроса

Некоторые типы вопросов имеют опции. Типы вопросов и значения опций приведены ниже.

  1. Несколько из нескольких Этот вариант Множественного выбора называется "Несколько из нескольких". Если функция "Опции вопроса" выбрана, то студент должен выбрать все правильные ответы из предложенных вариантов. В вопросе может говориться сколько правильных ответов имеет вопрос. Например вопрос "Кто из нижеследующих был президетом?" не говорит о количестве правильных ответов, в то время как "Выберите двух президентов из предложенного списка" говорит. Число правильных ответов может варироваться от одного до числа вариантов ответов. (Множественный выбор "Несколько из нескольких" с одним правильным ответом и Множественный выбор "Один из нескольких" совершенно два разных типа вопроса так как первый позволяет студенту выбрать более одного ответа а последний предусматривает выбор только одного ответа)

  2. Короткий ответ По умолчанию при сравнении ответа игнорируется регистр. Если Если функция "Опции вопроса" выбрана, то при сравнении ответа учитывается регистр.

    3. Другие типы вопросов не используют функцию "Опции вопроса".
