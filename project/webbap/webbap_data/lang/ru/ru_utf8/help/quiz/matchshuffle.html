Случайный выбор вопросов

Если вы выбираете "Да", то вопросы для каждой попытки прохождения теста будут показаны в случайном порядке.

Для того чтобы варианты ответов перемешивались внутри вопроса, необходимо в настройках теста "Случайный выбор ответа" выбрать "Да".

Это делается для того, чтобы ученики не списывали друг у друга.
