Типы Вики

Существует 3 типа Вики: Учитель, Группы, Ученик.
Как и все модули, Вики поддерживает режимы групп Moodle: "Нет групп"; "Отдельные группы" и "Доступные группы".

Вот таблица с 9-ю возможными вариантами:

  Нет групп Отдельные группы Доступные группы
Учитель Есть только одна Вики.
Редактировать ее может только учитель.
Ученик может просматривать содержимое.		 Есть только одна Вики для каждой группы.
Редактировать ее может только учитель.
Ученики могут просмотреть Вики только своей группы.		 Есть только одна Вики для каждой группы.
Редактировать ее может только учитель.
Ученики могут просмотреть Вики всех групп.
Группы Есть толька одна Вики. Учитель и ученики могут просматривать и редактировать. Есть одна Вики на группу. Ученики могут просматривать и редактировать Вики только своей группы. Есть одна Вики на группу. Ученики могут просматривать и редактировать Вики только своей группы, но просматривать Вики всех групп.
Ученик У каждого ученика своя Вики, которую только он и его учитель может просматривать и редактировать. У каждого ученика своя Вики, которую только он и его учитель может просматривать и редактировать. Ученики могут просматривать Вики других учеников своей группы. У каждого ученика своя Вики, которую только он и его учитель может просматривать и редактировать. Ученики могут просматривать Вики всех других учеников в курсе.

  • Примечание: Учитель всегда может редактировать все Вики в курсе.
