Часто задаваемые вопросы по модулю Книга

Почему только два уровня?
Два уровня обычно достаточно для всех книг. Три уровня обычно ухудшают структуру документа. Модуль Книга разрабатывался для создания небольших многостраничных учебных материалов. Для более объемных документов лучше использовать формат PDF. Наиболее простой способ создания PDF-файлов - использование виртуальных принтеров (например PDFCreator, PDFFactory (рекомендуем), Adobe Acrobat, и т.п.). Виртуальный принтер выглядит в Windows как обычный, но при печати на него вы получите свой документ в формате PDF.

Может ли ученик редактировать книги?
Только Учителя могут создавать и редактировать книги. Пока нет планов давать такую возможность ученикам. Основная причина - желание сохранить модуль Книга как можно более простым.

Могу ли я производить поиск внутри книг?
В настоящее время есть только один способ - поиск средствами браузера, используя страницу "для печати". Глобальный поиск пока возможен только в форумах системы. Было бы неплохо иметь возможность поиска во всех ресурсах, включая Книги. Есть добровольцы?

Названия глав не помещаются на одной строке содержания.
Попытайтесь перефразировать название или попросите администратора вашей системы Moodle увеличить ширину Содержания. Она определяется глобально для всех книг на странице конфигурации модуля.
