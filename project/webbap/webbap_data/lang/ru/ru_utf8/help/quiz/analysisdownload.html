Загрузка результатов анализа

Возможно, вы захотите скачать данные, отображаемые в экранной таблице, для дальнейшего изучения. Вы можете выбрать один из трёх форматов для загрузки данных
В каждом случае данные будут представлены в виде таблицы с такими же заголовками, как и на экране. Если таблица состоит из нескольких страниц, все данные со всех страниц будут включены в единый загружаемый файл.

Формат Таблицы Excel

Вы получите электронную таблицу в формате .xls.

Формат OpenOffice Writer

Данные будут представлены в виде таблицы в текстовом документе OpenOffice формата .sxw.
Этот формат доступен, только если в подкаталог /moodle/lib установлена библиотека PHPWriter.

Текстовый формат

В данном случае данные будут представлены в стандартном текстовом файле: каждой строке таблицы соответствует строка в файле; колонки разделены знаками табуляции.
