Загрузка результатов теста

вы можете загрузить данные, отображенные в таблице, для дальнейшего анализа. Вы можете выбрать между двумя форматами файла для загруженных данных.
В каждом случае данные будут представлены как таблица с соответствующими заголовками, как на экране. Если таблица будет состоять из нескольких страниц, то все данные для всех страниц будут загружены в отдельных файлах.

Электронная таблица формата Excel:

Вы получите электронную таблицу в документе .xls.

Текстовый формат

В этом случае, данные будут сохранены как обычный текстовый файл. Разделитель данных в каждой строке таблицы - табуляция.
