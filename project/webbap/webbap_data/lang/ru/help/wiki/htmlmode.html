Режим HTML

Вы имеете следующие возможности:

Не HTML:
Пропускаются все HTML теги. Форматирование - WikiWords. Фильтры в этом режиме работают.
Простой HTML:
Некоторые теги разрешены и отображаются. В этом режиме нет фильтра тегов.
Только HTML:
Нет WikiWords, только HTML. Эта опция позволяет использовать HTML-редактор.
