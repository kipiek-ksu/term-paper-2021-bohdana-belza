Показывать комментарий по умолчанию

Если установлено в Да, тогда, если не заданы комментарии для соответствующего вопроса, то будут использованый комментарии по умолчанию "Это правильный ответ" и "Это неправильный ответ".

Если установлено в Нет, тогда, если не заданы комментарии для соответствующего вопроса, то комментарии по умолчанию отображаться не будут. Пользователь будет автоматически перенаправлен к следующей странице лекции.
