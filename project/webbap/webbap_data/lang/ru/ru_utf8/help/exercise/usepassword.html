Использовать пароль

Обычно, если упражнение открыто, ученики могут видеть его содержимое. Доступность упражнения внутри курса лучше всего управляется настройкой Показать/Скрыть.

Опция "Использовать пароль" применяется для установки пароля доступа к упражнению. Бывают случаи, когда необходимо защитить упражнение паролем. Например, упражнение может быть задано нескольким группам учеников курса, для каждой группы установлено своё время выполнения упражнения. Пароль доступа используется, например, для того, чтобы предотвратить выполнение упражнения "второй" группой учеников в то время, как упражнение открыто для "первой" группы.
