Here you can enable the user-click frequency color system

If this feature is enabled, tags will be coloured by the number of users' clicks. The more the tag is red, the more the tag is popular.

Example

When this feature is not enabled, text Clouds will use only black color to write tags

footer html moodle document

Enabling the feature, every click on a tag will be counted, and each tag will be coloured based on the number of clicks received
