Отслеживание сообщений

Эта настройка предназначена для информирования участников форума о прочитанных или непрочитанных сообщениях.

Учитель может выбрать один из трех режимов отслеживания сообщений форума:

  • Не обязательно - ученики сами выбирают режим отслеживания в личных настройках;
  • Да - режим отслеживания включен;
  • Нет - режим режим отслеживания выключен.
