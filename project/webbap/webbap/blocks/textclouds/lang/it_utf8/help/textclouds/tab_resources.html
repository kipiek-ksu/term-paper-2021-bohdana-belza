In questo tab è possibile controllare e modificare lo stato di indicizzazione di ogni risorsa.

la tabella mostra la lista di ogni risorsa contenuta nel corso.

Id
Questo campo mostra l'Id di risorsa assegnato da Moodle

Nome
Questo campo contiene il nome della risorsa

Tipo
Questo campo contiene il tipo della risorsa. Può essere file, text, page o directory

Stato
Questo campo contiene lo stato di indicizzazione della risorsa. Può essere:
Pending: la risorsa è in attesa di essere indicizzata
Processed: la risorsa è stata indicizzata ed inclusa all'interno di Text Clouds
Disabled: la risorsa è stata disabilitata e non sarà indicizzata
Pending removal: la risorsa è in attesa di essere deindicizzata
Invisible: la risorsa non può essere indicizzata perchè invisibile. È possibile modificare la visibilità della risorsa all'interno del corso.
Unknown: la risorsa non può essere indicizzata perchè di tipo non supportato o sconosciuto.

Lingua
la lingua rilevata della risorsa

Includi
Selezione o deseleziona per includere o escludere la risorsa dall'indicizzazione

Gestisci
Quando una risorsa è stata indicizzata, è possibile cliccando sul link vedere i tags associati e modificarli.
