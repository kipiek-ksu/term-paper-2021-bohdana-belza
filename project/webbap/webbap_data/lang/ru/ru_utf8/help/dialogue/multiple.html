Многократные диалоги

Эта опция позволяет человеку начинать более одного диалога с кем-либо. Значение по умолчанию "Нет" - позволяет только один открытый диалог между двумя пользователями.

Разрешение многократных диалогов может привести к злоупотреблению этим средством. Некоторые люди могут надоедать другим, открывая с ними много нежелательных диалогов.
