How to Use this moodle theme

1. Upload the  folder into this directory: your-moodle-installation/theme/;

2. Go to  Administration -> Appearance -> Themes -> Theme Selector menu; 


How to Place your own logo

1: design your own logo and name it logo.jpg, upload to the theme pix folder and replace the current logo.jpg


How to adjust the top menu items?

1: Go to  Administration -> Appearance -> Themes -> Theme Settings

2: Follow the examples given and write your own menu
