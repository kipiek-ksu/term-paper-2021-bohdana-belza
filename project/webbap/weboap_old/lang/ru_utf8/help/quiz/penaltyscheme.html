Начисление штрафов

Тест может позволять студенту выполнять несколько попыток прохождения. В этом случае вы можете указать штраф, который будет вычитаться из итогового балла вопроса за каждый неверный ответ на него. Величина штрафа указывается индивидуально для каждого вопроса во время его создания или редактирования.

Это никак не влияет на тесты ограниченные единственной попыткой.
