Случайный вопрос на соответствие

После небольшого вступления ученику предлагается несколько вопросов и соответствуюшее число ответов. Для каждого из вопросов только один ответ является правильным.

Ученик должен выбрать для каждого вопроса соответствующий ему ответ.

Каждый вопрос автоматически имеет одинаковый вес.

Отличие данного типа от "Вопроса на соответствие" в том, что вопросы случайно выбираются из набора вопросов типа "Короткий ответ", находящихся в данной категории. При каждом прохождении теста выбираются различные вопросы. Количество вопросов можно регулировать.
