Роли

Роль - набор разрешений (прав), определенных для всей системы, которые вы можете назначить конкретным пользователям на определенных уровнях (сайт, категория, курс и т.д.).

Например, имеется роль "Учитель", которая установлена для пользователей - учителей и позволяет им выполнять определенные разрешенные действия (но не другие). Если эта роль существует, вы можете назначить эту роль кому-то в курсе, сделав его "Учителем" для этого курса. Вы можете также назначить эту роль пользователю в категории курсов, чтобы сделать его "Учителем" для всех курсов в этой категории, или назначить роль пользователю только на одном (отдельном) форуме, давая этому пользователю права только на конкретном форуме.

Роль должна иметь название (имя).

Описание содержит описание роли вашими словами, чтобы все имели общее представление об этой роли.

Смотрите также: Уровни, Права, Назначения Ролей и Исключения.
