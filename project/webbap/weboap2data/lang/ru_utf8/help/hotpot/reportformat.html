Формат отчета

вы можете определить различные форматы для отчетов Hot Potatoes.

Форматы отчетов

HTML:
отчет будет сгенерирован в HTML формате, доступном для просмотра в браузерах.

Excel:
этот отчет будет сгенерирован в ".xls" файле, который может быть открыт в Microsoft Excel.

Текст:
этот отчет будет сгенерирован в текстовом формате.

Кодированный

Если программа электронных таблиц требует определенной кодировки, то вы можете выбрать ее в выпадающем списке. Например, Microsoft Excel требует "SJIS" кодировку для японского языка. Список кодировок настраивается администратором сайта в модуле HotPot.

Обрезка значений

"Да"
Длинные значения данных будут обрезаться под размер ячеек таблицы.

"Нет"
Значения данных не будут обрезаться.

Показывать легенду

"Да"
в главном отчете вопросы и ответы будут маркированы буквами. Легенда отображает соответствие букв вопросов и ответов.

"Нет"
вопросы и ответы показываются полностью и легенды не будет.
