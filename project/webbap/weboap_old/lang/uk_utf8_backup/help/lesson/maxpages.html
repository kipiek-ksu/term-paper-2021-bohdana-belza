Количество страниц (карт) для показа

Этот параметр исползуется только для уроков типа Флэш-Карта. Значение по умолчанию - 0 что охзначает что ве страницы/карты показываются в уроке. Установка этого параметра в ненулевое значение указывает число страниц после которого будет достигнут конец урока и студенту будет показана оценка.

Если этот параметр установлен в значение превышающее число страниц урока, тогда конец урока будет достигнут после показа всех страниц.
