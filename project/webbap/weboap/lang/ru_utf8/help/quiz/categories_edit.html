Категории вопросов

Для удобной сортировки вопросов используются категории вопросов.

Каждая категория содержит имя и короткое описание.

Каждая категория также может быть "опубликована", это означает - категория (и все вопросы, содержащиеся в ней) станет доступна для всех курсов на сервере, значит в других курсах в тестах могут быть размещены вопросы из опубликованной категории

Описания полей:

Родительская категория: Категория, в которой будет размещен вопрос. Если нет других категорий, будет доступна только верхняя категория.

Категория: Имя категории.

Информация о категории: Краткое описание категории.

Опубликовать: Опубликовать категорию или нет.
