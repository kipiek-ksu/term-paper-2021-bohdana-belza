Несколько попыток

По умолчанию ученики не могут исправлять ответы после их оценки учителем.

Если вы включите эту опцию, ученикам будет разрешено высылать исправленные ответы после того, как они были оценены (для повторной оценки). Это может быть полезно в тех случаях, когда учитель хочет мотивировать учеников на подготовку еще более качественных ответов.

Очевидно, что этот параметр не может быть использован для заданий с ответом вне сайта.
