Дозвіл для нових повідомлень

У цьому полі Ви можете ввести обмеження для повідомлень студентів у форум.

Для більшості форумів Ви, швидше за все, будете використовувати першу опцію, щоб дозволити студентам починати нові теми (threads), і відповідати на інші повідомлення.

Іноді буває корисним використовувати третю опцію. Наприклад, у новостному форумі, що буде відображатися на першій сторінці курсу.

Можливо виникне ситуація, коли Вам знадобиться, щоб тільки вчителі могли відкривати нові теми, а учні - тільки відповідати. У цьому випадку доречно вибрати другу опцію - "Запитувати не можна, відповідати можна".
