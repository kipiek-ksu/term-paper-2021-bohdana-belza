Индивидуальные ответы

Эта таблица отображает как каждый пользователь отвечал на любой вопрос при попытках прохождения теста.

Вот цветовая схема ответов:

  • Правильные ответы
  • Проигнорированные ответы (если определены)
  • Неправильные ответы (если определены)
  • % задолжности (подсказки, ключи, проверки)
    • результат - процент задолжности для этого вопроса
    • подсказки - количество запрошенных подсказок
    • ключи - был ли показан ключ (=1) или нет (=0)
    • проверки - количество проверок, произведенных для этого вопроса. Значение "-1" говорит о том, что ученик нажал кнопку "Показать ответ"
