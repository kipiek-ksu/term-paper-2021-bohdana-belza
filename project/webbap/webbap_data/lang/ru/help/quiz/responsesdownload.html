Загрузка ответов

вы можете сохранить данные, отображенные на экране. Вы можете выбрать между двумя форматами файла для загрузки данных.

Если ответы будут состоять из нескольких страниц, то все данные для всех страниц будут загружены в отдельных файлах.Пользователи без попытки прохождения теста будут включены в файл, если вы включите соответствующую опцию.

Электронная таблица формата Excel:

Вы получите электронную таблицу в документе .xls.

Текстовый формат

В этом случае данные будут сохранены как обычный текстовый файл. Одна строка для каждого ряда значений в таблице; разделитель данных - табуляция.
