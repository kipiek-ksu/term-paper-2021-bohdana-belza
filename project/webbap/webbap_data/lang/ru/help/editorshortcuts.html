Горячие кнопки HTML редактора

Следующие действия доступны через комбинации горячих кнопок:

Ctrl-O Изменить шрифт
Ctrl-P Изменить размер шрифта
Ctrl-H Изменить формат текста
Ctrl-1 ... Ctrl-6 Изменить уровень заголовка
Ctrl-= Изменить язык текста, или определить язык для многоязыкового фильтра.
Ctrl-B Сделать текст жирным
Ctrl-I Сделать текст курсивом
Ctrl-U Подчеркнуть текст
Ctrl-S Зачеркнуть текст
Ctrl-, Нижний индекс
Ctrl-. Суперскрипт
Ctrl-0 Очистить HTML
Ctrl-Z Отмена
Ctrl-Y Вернуть
Ctrl-L Выровнять по левому краю
Ctrl-E По-центру
Ctrl-J По-ширине
Ctrl-/ Слева направо
Ctrl-| Справа налево
Ctrl-; Уменьшить отступ
Ctrl-' Увеличить отступ
Ctrl-G Изменить цвет текста
Ctrl-K Изменить цвет фона
Ctrl-F Найти и заменить
Ctrl-` Показать HTML код
Ctrl-M Переключить в полноэкраный режим
Ctrl-Alt-O Вставить маркированный список
Ctrl-Alt-U Вставить список
Ctrl-Alt-R Вставить горизонтальную линию
Ctrl-Alt-A Создать якорь
Ctrl-Alt-L Создать ссылку
Ctrl-Alt-D Отменить ссылку
Ctrl-Alt-N Отключить автоматическое связывание
Ctrl-Alt-I Вставить рисунок
Ctrl-Alt-T Вставить таблицу
Ctrl-Alt-S Вставить смайлик
Ctrl-Alt-C Вставить специальный символ

Текущие ограничения:

  • Вы можете использовать Ctrl-` для переключения в HTML-код и обратно.
