Количество страниц (карт) для показа

Этот параметр исползуется только для лекций типа Флэш-Карта. Значение по умолчанию - 0, что означает, что все страницы/карты показываются в лекции. Установка этого параметра в ненулевое значение указывает число страниц, после которого будет достигнут конец лекции и ученику будет показана оценка.

Если этот параметр установлен в значение превышающее число страниц лекции, тогда конец лекции будет достигнут после показа всех страниц.
