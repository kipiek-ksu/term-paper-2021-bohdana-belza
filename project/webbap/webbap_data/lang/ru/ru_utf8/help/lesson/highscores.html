Показать лучшие результаты

Отображение лучших для этой лекции результатов. Ученики, получившие высокие оценки, могут выбрать для себя имя в этом списке. Имя будет проверено фильтром 'badwords' для выявления неприемлемых значений.

Список лучших результатов не отображается, если включена опция 'Тренировочная лекция'.
