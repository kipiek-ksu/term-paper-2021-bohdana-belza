Ограничение времени

По умолчанию тесты не ограничены по времени и позволяют ученику использовать столько времени, сколько ему необходимо для завершения теста.

Если же вы ограничиваете время, то некоторые дополнительные возможности позволят вам быть уверенными, что тест завершен в течение этого времени:

  • Поддержка Javascript в браузере становится обязательной - это позволяет таймеру работать правильно.
  • В всплывающем окне отображается обратный отсчет времени.
  • Когда время заканчивается, результаты теста передаются на сервер автоматически вне зависимости от того на сколько вопросов получены ответы.
  • Если ученику удалось обмануть систему и потратить на 60 секунд больше времени, чем отведено, то попытка тестирования автоматически оценивается в ноль.
