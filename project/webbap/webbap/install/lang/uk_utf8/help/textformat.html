Форматування тексту

При написанні тексту на сайті, ви можете вибирати між двома форматами.

Не звертайте увагу на це поле, і все буде працювати так, як Ви й очікували.

1. Авто-формат

    Цей формат найбільше підходить, коли Ви не використовуєте Richtext HTML редактор для записів. Просто наберіть текст, як Ви це робите при відправленні e-mail.

    Після того, як Ви збережете текст, сайт виконає кілька речей для автоматичного форматування.

    Наприклад, web-адреси http://yahoo.com або www.yahoo.com будуть автоматично перетворені в посилання.

    Ваші переходи на новий рядок будуть збережені, а порожні рядки будуть починати нові параграфи.

    "Смайли" :-) будуть автоматично перетворені в графічні еквіваленти.

    Ви можете вставляти html-теги для додаткового форматування.

    Інформація про авто-формат

2. HTML-формат

    Цей формат припускає, що текст є чистим HTML. Якщо Ви використовуєте HTML редактор для редагування тексту, то цей формат буде встановлений за замовченням.

    Навіть якщо Ви не використовуєте Richtext HTML редактор, використавши HTML код у тексті, Ви отримаєте на виході те, що хотіли.

    На відміну від авто-формату, ніякого автоматичного перетворення не відбувається.
