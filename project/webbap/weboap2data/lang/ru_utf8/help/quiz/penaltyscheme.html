Начисление штрафов

Тест может быть запущен в обучающем режиме: это означает, что ученики смогут несколько раз отвечать на каждый вопрос. В этом случае вы можете указать штраф, который будет вычитаться из итогового балла вопроса за каждый неверный ответ на него. Величина штрафа указывается индивидуально для каждого вопроса во время его создания или редактирования.

Эта настройка имеет значение, только если тест запущен в обучающем режиме.
