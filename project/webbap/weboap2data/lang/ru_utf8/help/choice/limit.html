Ограничение выбора варианта ответа

Эта опция позволяет включить режим ограничения числа учеников, которые могут выбрать конкретный вариант ответа.

Когда режим включен, то в поле "Предел" нужно установить это число. Когда при опросе установленное количество учеников выбрали этот вариант ответа, то для остальных учеников он будет закрыт. Если установлено 0, то вариант ответа будет недоступен.

Если режим выключен, то любое число учеников может выбрать конкретный вариант ответа на поставленный вопрос.
