Вопросы на соответствие

После небольшого вступления студенту предлагаются несколько подвопросов и соответствуюшее число ответов. Для каждого из вопросов только один ответ является правильным.

Студент должен выбрать для каждого подвопроса соответствующий ему ответ.

Каждый подвопрос автоматически имеет одинаковый вес.
