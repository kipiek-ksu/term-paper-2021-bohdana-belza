Неоцененные оценки (Работ учеников)

Оценка работ учеников. Это оценки, сделанные учениками о работах друг друга. Вообще, эти оценки не должны оцениваться учителем. Если каждая студенческая работа оценена около пяти раз, система может принять приемлемое решение по оценке индивидуальной работы ученика. Когда количество оценок мало, тогда учитель должен оценивать эти оценки. Все оценки учителя, данные этим оценкам, будут учтены при вычислении итоговых оценок для учеников.
