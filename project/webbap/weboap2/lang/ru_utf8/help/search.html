Как искать

Полнотекстовой поиск поддерживает опции, перечисленные ниже. Вы можете комбинировать их для лучших результатов поиска:

search for Для поиска одного или нескольких слов в тексте, просто напечатайте их, разделяя пробелами. Используются любые слова длиннее двух символов.
+search +for Предыдущий пример найдет слово "информация", если искать "форма". Для более тщательного поиска используйте знак плюс "+".
+search -engine Используйте знак "-" если не хотите включать части слов в результаты поиска.
"search engine" Для поиска целой фразы, просто заключите ее в кавычки.
user:Kim Для поиска текста определенного пользователя, напишите перед именем пользователя префикс "user:".
userid:6 Если вы знаете Id определенного пользователя, вы можете найти его таким способом.
subject:assessment Если вы ищите текст внутри какого-то раздела, поставьте префикс "subject:".

Для более расширенного поиска, нажмите кнопку поиска без написания слов в поле - вы увидите форму, которая сделает поиск удобней и эффективней.
