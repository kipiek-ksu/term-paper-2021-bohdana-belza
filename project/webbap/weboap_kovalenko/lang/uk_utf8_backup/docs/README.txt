Документация.
--------------------

Если Вы хотите перевести систему на другой язык,
я бы Вам посоветовал:

 1) не включайте английские версии непереведенных файлов, так как они в любом случае будут использоваться автоматически.

 2) переводите в следующем порядке (по шкале):
  
   НАИБОЛЕЕ ВАЖНЫЕ
    |-------------------
    |
    |   - files.php
    |   - install.html
    |   - upgrade.html
    |
    |  -----------------
    |
    |   - teacher.html
    |   - module_files.txt
    | 
    |  -----------------
    |   
    |   - intro.html
    |   - features.html
    |   - release.html
    |
    |  -----------------
    |
    |   - developer.html
    |   - cvs.html
    |   - future.html
    |
    |  -----------------
    |   
    |   - license.html
    |   - credits.html
