Файлы к сообщениям

Вы можете добавить один файл(картинку, Word-документ...) к своему сообщению в форум. Файл будет храниться на сервере вместе с Вашим сообщением.

Файл может быть любого типа, но мы советуем использовать стандартные 3-х буквенные расширения для файлов (например, .doc для Word-документов, .jpg или .png для картинок...), для того чтобы файлы было проще идентифицировать другим.

Если Вы изменили свое сообщение и добавили новый файл, предыдущий файл будет заменен.

Если Вы отредактировали сообщение и оставили поле с прилагаемым файлом пустым, предыдущая версия файла будет использоваться с сообщением.
