Допомога в написанні HTML

При написанні HTML на сайті, Ви можете використовувати будь-які HTML-теги, для створення потрібних Вам ефектів.

Звичайно Ваш код буде розміщений в комірці таблиці, у такий спосіб:

  • не потрібно використовувати теги <HEAD> або <BODY>;
  • будьте уважні з використанням тегу </TABLE>, що може зламати структуру сторінки сайту.
