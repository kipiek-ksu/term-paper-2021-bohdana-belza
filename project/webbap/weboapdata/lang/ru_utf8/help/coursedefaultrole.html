Роль по умолчанию

Эта роль используется, если никакая роль не назначена; например, когда люди саморегистрируются в вашем курсе, используя пароль.

Она также используется как значение по умолчанию для методов регистрации участников этого курса, основанного на данных от других систем.

Для большинства случаев эта опция установлена по умолчанию. Администратор сайта может назначить роли по умолчанию для всех курсов.
