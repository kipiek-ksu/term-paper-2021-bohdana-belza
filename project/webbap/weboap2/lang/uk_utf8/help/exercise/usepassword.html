Использовать пароль

Обычно, если упражнение открыто, студенты могут видеть содержимое упражнения. Доступность упражнения внутри курса лучше всего управляется настройкой Показать/Скрыть.

Опция "Использовать пароль" применяется для установки пароля "доступа" к упражнению. Бывают случаи, когда необходимо защитить упражнение паролем. Например, упражнение может быть задано нескольким группам студентов курса, для каждой группы установлено своё время выполнения упражнения. Пароль доступа используется, например, для того, чтобы предотвратить выполнение упражнения "второй" группой студентов в то время, как упражнение открыто для "первой" группы.
