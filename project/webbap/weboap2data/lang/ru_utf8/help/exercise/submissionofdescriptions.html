Отправка описания Упражнения

Учитель должен отправить по меньшей мере одно описание задания или упражнения. Описание может содержаться в документе Word или в HTML-файле (или любом другом типе файлов, который может нормально отображаться в браузере). Этот файл будет показан ученикам. Он должен содержать инструкции по выполнению упражнения.

Учитель может отправить несколько описаний упражнения. В таком случае разные ученики будут видеть разные варианты упражнения, выбираемые случайным образом из загруженных учителем описаний. Варианты должны быть похожи, т.к. для оценки работы используется одна и та же Форма оценки.
