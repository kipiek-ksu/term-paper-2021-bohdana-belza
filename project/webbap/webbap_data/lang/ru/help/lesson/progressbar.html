Индикатор выполнения

Показать Индикатор выполнения на страницах лекции. На данный момент иникатор выполнения корректно работает только для лекций с линейной последовательностью страниц.

При подсчете используется количество пройденных учеником страниц и правильно отвеченных вопросов. Кроме того, кластеры и страница в них считаются как одна страница, конец кластеров и разделов не учитывается.

Примечание: стиль используемый по умолчанию недостаточно впечатляющий ;) Любой его параметр можно определить в файле mod/lesson/styles.php.
