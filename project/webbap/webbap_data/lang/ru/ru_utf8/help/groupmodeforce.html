Принудительный групповой режим

Если эта настройка включена, то выбранный групповой режим применяется ко всем элементам данного курса. Изменение режима для отдельных элементов курса становится невозможным.

Это полезно, например, если необходимо создать курс для нескольких полностью отдельных групп.
