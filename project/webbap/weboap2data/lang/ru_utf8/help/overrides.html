Исключения

Исключения специфических разрешений используются, чтобы отвергнуть роль в конкретном контексте, позволяя вам "тонко настроить" требуемые разрешения.

Например, пользователи с ролью Ученик обычно могут начинать новые темы в форумах, но есть один специфический форум, для которого вы хотите ограничить эту возможность, тогда вы можете установить Исключение, которое ПРЕДОТВРАЩАЕТ разрешение для Учеников "Начать новые темы".

Исключения могут также использоваться, чтобы "открыть" области вашего сайта и курсов для предоставления пользователям добавочных разрешений там, где это имеет смысл. Например, вы хотите экспериментально предоставить Ученикам несколько дополнительных разрешений.

Интерфейс подобен назначению ролей, иногда показаны только существенные возможности, и вы будете видеть несколько разрешений, выделенных, чтобы показать вам, какие разрешения были бы у роли БЕЗ этого Исключения (в случае, когда эта роль УНАСЛЕДОВАНА).

Смотрите также: Роли, Уровни, Назначения ролей и Права.
