In questo tab si possono definire quali parole devono essere ammesse o rimosse nell'indicizzazione. Queste parole saranno aggiunte o rimosse dalla lista di stopwords. La lista di stopwords è specifica per lingua, ma le modifiche effettuate in questo tab verranno applicate a tutte le lingue e rimarranno contestualizzate all'interno del proprio corso.

Parole da ammettere

Questa lista contiene le parole che devono assolutamente essere accettare in fase di indicizzazione. Se si hanno dei dubbi che qualche termine desiderato venga bloccato in fase di indicizzazione,si può inserire in questa lista e, anche se presenti nella lista delle stopwords, verranno comunque prese in considerazione.

Parole da bloccare

Questa lista contiene le parole che devono assolutamente essere escluse in fase di indicizzazione. Questa lista verrà aggiunta a quelle delle stopwords in qualsiasi linguaggio.

Come modificare le liste

Per aggiungere una parola ad una delle liste, inserirla nell'area di testo in fondo alla lista desiderata e premere "aggiungi". Per rimuovere una parola, premere il pulsante "rimuovi" che si trova accanto alla parola da rimuovere. Se si inserisce una parola presente nell'altra lista, ad esempio aggiungo una parola da ammettere che si trova nella lista di quelle da bloccare, essa verrà automaticamente rimossa dalla vecchia lista e aggiunta nella nuova.
