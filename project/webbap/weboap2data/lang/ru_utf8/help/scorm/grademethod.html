Методы оценивания

Результаты обучения для Пакета SCORM/AICC, показываемые на страницах Оценок, могут быть оценены несколькими способами:

  • Объекты обучения:
    Этот способ показывает количество законченных/переданных Объектов обучения. Максимальное значение - число Пакетов Объектов обучения.
  • Высшая оценка:
    На странице оценок будет показан самый высокий результат, полученный пользователем из всех переданных Объектов обучения.
  • Средняя оценка:
    Если вы выбираете этот способ, Moodle вычислит среднее значение из всех результатов.
  • Суммарная оценка:
    Для этого способа все результаты будут суммироваться.
