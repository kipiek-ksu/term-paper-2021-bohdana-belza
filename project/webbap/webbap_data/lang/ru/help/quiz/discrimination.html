Индекс дискриминации

Представляет из себя достаточно грубый показатель качества каждого вопроса. Все ученики делятся на три группы по рейтингу, набраному в тесте. Индекс каждого вопроса находится делением количества учеников из верхней трети, правильно ответивших на вопрос, на количество учеников из нижней трети, тоже ответивших правильно.

Например, если 30 учеников прошли тест, то по 10 учеников окажется в верхней и нижней третях. Если на вопрос №1 правильно ответили 9 учеников из лучшей группы и только 3 из худшей, то индекс дискриминации будет равен 9/3 = 3.0. Числа в скобках: (количество верных ответов из верхней группы/количество верных ответов из нижней группы).

Если индекс оказывается меньше 1.0, то это означает что слабые ученики лучше отвечали на вопрос, чем сильные. Такие вопросы должны быть исключены из теста, как ничего не стоящие. Фактически, они уменьшают точность общей оценки теста.

Если ни один ученик из слабой группы не ответил правильно на вопрос, то знаменатель дроби оказывается равным 0, что дает результат, равный бесконечности. Программа делает индекс такого вопроса = 10.
