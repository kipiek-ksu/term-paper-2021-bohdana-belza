Содержание отчета

Вы можете определить содержание для отчетов Hot Potatoes.

Выбор типа отчета

Общий просмотр
Этот отчет показывает список всех попыток в тесте. Он показывает фотографию и имя каждого пользователя, оценку за тест, длительность каждой попытки. Попытки могут быть выбраны или удалены, если это требуется.

Простая статистика
Этот отчет показывает подробные детали всех завершенных попыток в тесте. Он показывает имя пользователя, его фотографию, оценку за тест и оценку за каждую попытку. Отображаются результаты по каждому вопросу.

Подробная статистика
Этот отчет показывает полную статистику всех завершенных попыток в тесте. Этот отчет содержит следующие таблицы:

  • Таблица ответов показывает как каждый пользователь ответил на каждый вопрос в каждой попытке теста.
  • Таблица анализа вопроса показывает список ответов на каждый вопрос и частоту каждого ответа.

Отчет по щелчкам
Этот отчет показывает информацию о каждом щелчке ученика в каждой попытке теста, исключая незавершенные. Отчет очень большой, так что его лучше всего просматривать в Microsoft Excel. Детальный отчет о столбцах таблицы доступен в файле помощи Таблица результатов щелчков.

Отчет по щелчкам доступен только в том случае, если он разрешен для этого теста HotPot.

Выбор пользователей

Все пользователи:
Этот отчет отображает попытки всех пользователей пройти тест. Он включает всех пользователей, зарегистрированных в курсе; гостей, создателей курсов и администраторов Moodle.

Все участники:
Этот отчет показывает все ответы и попытки учеников и учителей курса.

Группа: название группы
Этот отчет показывает все обращения выбранной группы. Группы не будут доступны в меню, если в курсе нет зарегистрированных групп.

Зарегистрированные ученики:
Этот отчет показывает все ответы зарегистрированных учеников. Ответы учителей не включаются.

Отдельный ученик:
Этот отчет показывает ответы отдельного ученика, выбранного из выпадающего списка.

Выбор оценок

Все попытки:
Этот отчет показывает ответы учеников во время всех попыток.

Лучшая попытка:
Этот отчет показывает ответы только из лучшей попытки каждого пользователя.

Первая попытка:
Этот отчет показывает ответы только из первой попытки каждого пользователя.

Последняя попытка:
Этот отчет показывает ответы только из последней попытки каждого пользователя.
