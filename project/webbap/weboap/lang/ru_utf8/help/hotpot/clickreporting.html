Разрешить учет щелчков

Да
Отдельная запись сохраняется в базе данных каждый раз при нажатии кнопок "Подсказка", "Ключ" или "Проверка". Это позволяет учителю просмотреть очень подробный отчет, показывающий состояние теста при каждом щелчке. Однако, это также значит, что размер базы данных очень быстро растет, потому что несколько записей "щелчков" могут быть сохранены для каждой попытки прохождения теста.

Нет
На каждую попытку прохождения теста в базе данных Moodle сохраняется только одна запись.

По умолчанию эта опция отключена.
