Отправка описания Упражнения

Учитель должен отправить по меньшей мере одно описание работы или упражнения. Описание может содержаться в документе Word или в HTML-файле (или любом другом типе файлов, который может нормально отображаться в браузере). Этот файл будет показан студентам. Он должен содержать инструкции по выполнению упражнения.

Учитель может отправить несколько описаний упражнения. В таком случае разные студенты будут видеть разные варианты упражнения, выбираемые случайным образом из загруженных учителем описаний. Варианты должны быть похожи, т.к. для оценки работы используется одна и та же Форма оценки.
