Результаты щелчков

Эта таблица показывает детали каждого щелчка всех учеников на протяжении завершенных попыток конкретного теста Hot Potatoes.

Отметим, что отчет очень большой, он может быть загружен и просмотрен в табличном редакторе, например Microsoft Excel.

Каждая строка в таблице содержит следующую информацию:

Какой щелчок?
Эти строки показывают следующие детали: какой щелчок в какой попытке ученика в тесте в определенной секции курса:

  • Название курса, Номер секции, Номер упражнения, Название упражнения, Тип упражнения, Количество вопросов
  • Полное имя ученика, Дата входа, Время входа, Время выхода
  • Номер попытки, Время начала попытки, Время завершения попытки
  • Номер щелчка, Время щелчка, Тип щелчка

Какие вопросы были затронуты?
Следующая информация показывает, какие вопросы были затронуты щелчками ученика:

  • вопрос был проверен, или текущим щелчком был получен ключ или подсказка
  • статус каждого вопроса после щелчка
    • 0 : "Верно"
    • X : "Ошибочно"
    • - : "без попытки"
  • новые ответы, которые были получены во время этого щелчка (ответы, которые не изменились, не отображаются)
  • вопросы, на которые были получены ответы во время этого щелчка
  • вопросы, на которые во время этого щелчка были получены ответы, подсказка или ключ

Сколько вопросов было затронуто?
Следующая информация показывает общее количество вопросов, затронутых текущим щелчком, или количество вопросов, затронутых недавно:

  • количество вопросов с попытками
  • количество вопросов без попыток
  • количество вопросов со статусом "Верно"
  • количество вопросов со статусом "Ошибочно"
  • количество вопросов, для которых были получены подсказки
  • количество вопросов, для которых были получены ключи
  • количество вопросов, для которых были получены новые ответы
  • количество вопросов, для которых были получены ответы, подсказки или ключи

Результаты
Следующая информация показывает результаты этого щелчка и недавнего теста:

  • необработанный результат
  • максимальный результат
  • результат в процентном соотношении
