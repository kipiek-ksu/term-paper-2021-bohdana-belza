Обработка нескольких попыток

Данная опция определяет, как будут рассчитываться итоговые оценки. Когда есть несколько вариантов выполненной работы, учитель может выбрать, использовать ли наилучшую оценку или среднюю оценку из всех вариантов, присланных учеником. Эту опцию можно изменять в любое время. Любое изменение немедленно отражается на странице Оценок (конечно, если есть несколько вариантов работы).
