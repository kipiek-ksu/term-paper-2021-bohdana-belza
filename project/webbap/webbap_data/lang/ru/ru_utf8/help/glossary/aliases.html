Ключевые слова

Каждый термин в глоссарии может быть ассоциирован с набором ключевых слов (псевдонимов).

Такие слова могут использоваться для альтернативного поиска, а также при создании автоматически создаваемых связей между словами в текстах курса и в статье глоссария.

Пример: Для термина Траектория - (линия, по которой движется тело)- ключевыми словами будут: путь, линия движения.
