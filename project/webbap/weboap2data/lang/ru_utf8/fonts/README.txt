Шрифт
-----

В этой директории находится шрифт, который будет использоваться при выводе графиков.

Используемый шрифт должен называться - default.ttf

Если в этой папке нет шрифта, система использует по-умолчанию
/lang/en/fonts/default.ttf.

Мультибайтовые строки требуют перекодировки. За это должен отвечать 
файл с названием lang_decode.php, содержащий функцию lang_decode().
