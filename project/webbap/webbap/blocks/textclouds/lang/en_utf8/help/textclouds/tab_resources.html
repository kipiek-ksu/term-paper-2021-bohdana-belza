In this tab you can check the status of each resource indexing.

The table shows the list of course resources with the indexing status.

Id
This field shows the Moodle resource Id

Name
This field shows the resource name

Type
This filed shows the resource type. Can be file, text, page or directory

Status
This field show resource indexing status. Can be:
Pending: resource is waiting to be indexed
Processed: resource has been indexed and included in Text Clouds
Disabled: resource has been disabled and is not indexed
Pending removal: resource is waiting to be deindexed
Invisible: resource can't be indexed because is hidden. You can set visibility in editing mode in the course
Unknown: resource can't be indexed because is of an unknown or unsupported file format

Language
The language detected for the resource

Include
Check or uncheck a resource for add/remove it from indexing

Manage
When a resource is indexed, click on manage to show tags associated. You can add, edit or delete tags.
