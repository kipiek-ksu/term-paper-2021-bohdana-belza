Ограничение по времени

Эта установка определяет предельную продолжительность лекции. Ученики будут видеть таймер на javaScript и потраченное ими время будет записано в базу данных. Из-за особенностей javaScript таймер не выбросит ученика в случае превышения времени, тем не менее ответы на вопросы после лимита времени не будут учтены. Ограничение по времени проверяется в базе данных каждый раз, когда ученик отвечает на вопрос.
