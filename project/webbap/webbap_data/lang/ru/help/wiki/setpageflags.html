Установка флагов страницы

Флаги назначаются страницам Вики и позволяют менять тип их поведения.

Описание флагов:

TXTСтраница содержит текст.
BINСтраница содержит бинарные данные.
OFFСтраница отключена.
HTMLРазрешить HTML для этой страницы (настройка Вики предшествует этому флагу).
ROТолько для чтения.
WRДля записи.
