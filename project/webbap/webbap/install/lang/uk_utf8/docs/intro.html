Система МООДУС (Модульна Об'єктно - Орієнтована Навчальна Система) є пакетом програмного забезпечення для створення курсів дистанційного навчання й web-сайтів. Цей проект, що продовжує розвиватися, був створений для підтримки й досліджень теорії "social constructionist framework of education".

Система поширюється безкоштовно, як Open Source-проект, по ліцензії GNU GPL (кількома словами: Вам дозволяється копіювати, використовувати й змінювати код).

Система може працювати на будь-якому комп'ютері, на якому працює PHP, і підтримує практично будь-яку базу даних (зокрема, MySQL).
