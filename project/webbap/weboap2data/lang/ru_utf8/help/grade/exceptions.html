Исключения

Данная функциональность может использоваться, чтобы НЕ оценивать выполнение определённых заданий конкретными учениками. Это полезно при объединении двух подгрупп или при переводе ученика из другой подгруппы в течение семестра. Это также полезно для учёта смягчающих обстоятельств: болезни, травмы и т.п.
Есть три колонки:

  • Левая: список учеников курcа, выполнение которыми конкретного задания оценивается.
  • Средняя: список всех оцениваемых заданий, после названия задания в скобках указано общее число учеников, чьи действия в данном случае НЕ оцениваются.
  • Правая: список учеников, чья работа по конкретному заданию НЕ оценивается

      • Чтобы исключить ученика из списка оцениваемых, выберите задание в средней колонке, затем выберите имя ученика в левой колонке (удерживайте CTRL или APPLE, чтобы выбрать несколько элементов). После этого нажмите на кнопку "НЕ оценивать", расположенную под левой колонкой. Ученик(ы) будет перемещен из левой колонки в правую, теперь он будет исключён из подсчёта оценок для этого задания.

      Чтобы вернуть ученика в список оцениваемых, выберите нужное задание, выберите ученика в правой колонке и нажмите на кнопку "Оценивать", расположенную под правой колонкой. Ученик будет перемещён из правой колонки в левую.
