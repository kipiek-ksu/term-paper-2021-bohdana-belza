Множественный выбор

Ученик выбирает ответ на вопрос из нескольких предложенных ему вариантов, причем вопросы могут предполагать один или несколько правильных ответов.

Оценки за вопросы с одним правильным ответом должны быть положительными.

Оценки за вопросы с несколькими правильными ответами должны быть как положительными, так и отрицательными (для того, чтобы, выбрав все варианты, ученик не получил положительную оценку). Если после ответов оценка - отрицательная, результат обнуляется. Будьте осторожны, чтобы не создать возможность получения оценки больше чем 100% (следите за суммой).

Каждый вариант ответа может содержать комментарий, который будет показан напротив выбора ученика после окончания теста (если это было разрешено при создании теста).
