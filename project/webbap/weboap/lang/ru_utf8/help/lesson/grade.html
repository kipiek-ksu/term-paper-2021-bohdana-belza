Оценка за лекцию

Этот параметр определяет максимальную оценку, которая может быть получена за лекцию в диапазоне от 0 до 100%. Это значение может быть изменено в любое время. Любые изменения незамедлительно оказывают влияние на страницу "Оценки" и на оценки отображаемые ученикам. Если значение параметра установлено в 0, то лекция становится невидимой на различных страницах содержащих оценки.
