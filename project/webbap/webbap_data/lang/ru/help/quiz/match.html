Вопросы на соответствие

После небольшого вступления ученику предлагаются несколько вопросов и соответствуюшее число ответов. Для каждого из вопросов только один ответ является правильным.

Ученик должен выбрать для каждого вопроса соответствующий ему ответ.

Каждый из вопросов автоматически имеет одинаковый вес.
