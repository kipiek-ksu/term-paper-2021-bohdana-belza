Количество приложений к работе

Число, введенное здесь, определяет сколько полей "загрузить" будет доступно ученику, представляющему свою работу. Число может быть 0, тогда приложения к работе не разрешены. Если приложения ожидаются, число может быть 1, 2... до 5. Обычно устанавливается значение 0 или 1, но в некоторых заданиях учеников можно попросить представить больше одного приложения.

Если, допустим, установлено значение 3, а ученик прилагает только два файла к своему представлению, он не получит никакого предупреждающего сообщения. Таким образом, представляя работу, ученики могут приложить произвольное количество файлов к своему представлению до максимального значения, заданного этим выбором (за один раз).

Отметим, что эта опция не определяет максимальное число приложений, которые ученик может добавить к своей работе. Она определяет, сколько полей "загрузить" будет показано. Ученик может добавить больше приложений к своему представлению, редактируя его. Это бывает крайне редко.

Значение по умолчанию для этой опции - 0, т.е приложения не требуются.
