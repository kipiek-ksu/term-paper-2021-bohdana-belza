Количество знаков после запятой

С помощью этого параметра вы можете выбрать, сколько знаков после запятой будет отображаться ученику в баллах и оценке. Например, значение "0" означает, что оценка будет округлена до целого.

Этот параметр используется только для отображения оценок. Внутренние расчеты не зависят от этого параметра.
