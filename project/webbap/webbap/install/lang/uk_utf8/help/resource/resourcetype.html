Типи ресурсів

Текст. Ви можете друкувати або редагувати текст у формі на наступній сторінці. При цьому буде відбуватися автоформатування.
ще...

HTML-текст. Автоматичне форматування тексту не відбувається. Форматування тексту робить сам користувач, за допомогою HTML-тегів.

Посилання - посилання на статтю в журналі або книгу.

Заколисаний файл - відображає будь-який файл, що Ви захитали для курсу. Див. розділ 'Файли'.

Web-посилання - URL до чого-небудь в web.

Web-сторінка - Працює аналогічно Web-посиланню, за винятком того, що ресурс відкривається у фреймі (інтегрується із сайтом).

Програма - зовнішня програма, що на вході приймає інформацію про системного користувача й курс.
