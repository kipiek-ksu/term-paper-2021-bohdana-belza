Видимость поля Profile

Каждому полю можно дать один из трех параметров настройки видимости: "Не видно", "Видно всем" и "Видно пользователю".

Установка "Не видно" обычно устанавливается администратором для частных данных пользователей.
"Видно пользователю" обычно выбирается для поля, которое содержит личную информацию пользователя.
Установка "Видно всем" может использоваться для открытого типа информации.
