Максимальный размер загружаемого файла

Эта установка определяет максимальный размер файла который может загрузить студент в пределах курса. Она не может быть больше значения, установленного администратором для всего сайта.

Вы можете в последствии уменьшить это значение для отдельных элементов вашего курса.
