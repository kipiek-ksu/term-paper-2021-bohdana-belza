Название первой страницы Вики

Здесь вы можете задать название первой страницы Вики, если хотите, чтобы оно отличало Вики. Это название станет первой страницей Вики для каждого последующего Вики.

Если вы оставите это поле пустым, то названием первой страницы будет название Вики (если вы странице не зададите имя в поле "первая страница").
