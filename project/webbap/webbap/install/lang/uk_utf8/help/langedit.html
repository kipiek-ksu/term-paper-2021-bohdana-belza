Редагування перекладу

Для редагування перекладу на цій сторінці, файли на сервері не повинні мати атрибута "тільки для читання".

У деяких рядках Ви побачите $a або $a->something.

Ці символи показують позиції в реченні, де будуть знаходиться змінні.

Коли в реченні тільки одна змінна, використовується форма - $a. Коли змінних дві й більше, кожній надається ім'я.
