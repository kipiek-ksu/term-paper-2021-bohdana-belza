Размещение полей

Это меню позволяет вам сохранять данные из существующей базы данных. Чтобы сохранить данные в новом поле, вы должны выбрать значение в новом поле из списка данных. Любое поле можно оставить пустым, без информации. Любое старое поле, не отображенное в новом, будет потеряно и все данные удалены.

Вы можете выбирать поля только того же самого типа, таким образом каждое всплывающее поле будет иметь различные поля. Не пытайтесь отображать одно старое поле к более, чем одному новому полю.
