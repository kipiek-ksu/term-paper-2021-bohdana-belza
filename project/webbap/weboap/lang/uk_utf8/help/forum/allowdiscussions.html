Разрешения для новых сообщений

В этом поле Вы можете ввести огрничения для сообщений студентов в форум.

Для большинства форумов Вы, скорее всего, будете использовать первую опцию, чтобы разрешить студентам начинать новые темы (threads), и отвечать на другие сообщения.

Иногда бывает полезным использовать третью опцию. Например, в новостном форуме, который в последствии будет отображаться на первой странице курса.

Возможно возникнит ситуация, когда Вам понадобится, чтобы только учителя могли открывать новые темы, а ученики - только отвечать. В этом случае уместно выбрать вторую опцию - "Спрашивать нельзя, отвечать можно".
